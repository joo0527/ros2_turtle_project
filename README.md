ROS 2 기반 가상 Lidar 데이터 파이프라인 및 분석

이 프로젝트는 실제 로봇 하드웨어 없이도 **가상의 Lidar 센서 데이터(360개 방향 거리 정보)**를 생성하고, 이를 ROS 2 통신을 통해 실시간으로 수집 및 분석하는 시스템을 구축한 프로젝트입니다.

*주요 특징
- **Virtual Data Generation:** 터틀봇3의 Lidar 센서 데이터를 모사한 가상 데이터를 생성하여 통신 테스트를 진행했습니다.
- **Data Engineering:** 생성된 가상 데이터를 MySQL DB에 실시간으로 적재(Ingestion)하는 파이프라인을 구현했습니다.
- **Scalability:** 실제 로봇 연결 시에도 코드 수정 없이 바로 적용 가능한 구조로 설계되었습니다.

*주요 파일 설명
- `move_decision.py`: 가상 Lidar 데이터를 수신하여 로봇의 가상 주행 결정을 내리고, 해당 시점의 센서 정보를 MySQL에 저장합니다.
- `data_analysis.py`: DB에 쌓인 가상 데이터를 불러와 데이터의 정규성이나 분포를 확인하는 분석 코드입니다.

*프로젝트 목표
- ROS 2의 토픽(Topic) 통신 구조 이해
- 대량의 센서 데이터(360 columns) 처리 및 DB 연동 실습
- Pandas를 활용한 시계열 데이터 전처리 기술 습득
